## Day2（2026/03/02）
### 🎯 目的
CSSの基礎を身に着ける
### ✅ 実装内容
- 基本装飾（フォント、色、ボーダーなどの装飾）
- レイアウト調整（display:flex、justify-content）
### 🔄 主な修正
- text-align → widthとmargin: 0 auto;を追加
### 💡 学び
- flexは常に親にかける
- flexコンテナは親の横幅100％のサイズになる
- text-alignはインライン要素かinline-blockにのみ効く
### 🚧 課題
- 親子関係の理解が曖昧
- align-itemsを使えるようになる
- 「なぜこうしたか？」が言語化できない

## Day3（2026/03/03）
### 🎯 目的
CSSの基礎を身に着ける
### ✅ 実装内容
- Day2のレイアウト見直し

- HTMLの調整・整理
- キャッチフレーズ、商品について、スタッフ紹介、アクセス
### 🔄 主な修正
- classとCSSの連携を調整
- h2にspanを追加

### 💡 学び
- ブロック要素とflex要素は別もの（flexアイテムにtext-alignは効かない）

- やりたいことを言語化すると推測が立てやすくなる
### 🚧 課題
- flexアイテムとブロック要素の理解を深める
